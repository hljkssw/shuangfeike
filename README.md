# 双飞客(shuangfeike)

## 关于双飞客

双飞客官方网站的服务器采用CentOS8.1+Apache2.4+PHP7.2+MySQL8.0配置，在配置服务器时使用的CentOS默认yum安装，在框架上使用的是CodeIgniter 4开发，主要的功能有新闻文章、会员中心、订单系统等，后期在根据实际需要，再不断增加。

欢迎访问 [官方网站](http://www.shuangfeike.cn)，也欢迎使用[开发库](https://github.com/hljkssw/shuangfeike)。
